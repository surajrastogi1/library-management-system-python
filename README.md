# 📚 Library Management System (Python)

A simple Library Management System built using Python and Object-Oriented Programming (OOP).  
This project demonstrates core programming concepts such as classes, objects, functions, and data management.

## 🚀 Features

- Add new books to the library
- View available books
- Borrow books
- Return books
- Manage book records

## 🧠 Concepts Used

- Python Fundamentals
- Object-Oriented Programming (OOP)
- Classes and Objects
- Functions
- Basic Data Structures

## 🛠 Technologies Used

- Python 3


## ▶️ How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the program

## 🎯 Learning Outcome

This project helped in understanding:
- Object-Oriented Programming in Python
- Managing data using classes
- Building a simple real-world application
